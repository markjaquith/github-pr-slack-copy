Features

Draft-aware
  • Draft pull requests use :draft:
  • Everything else uses :pr:
Rich text
  • Copies rich text
  • Falls back to markdown for plain-text pasting

Installation and use

  1. Drag Copy PR for Slack to your bookmarks bar
  2. Open any pull request on github.com
  3. Click the bookmark to copy the formatted link
  4. Paste into Slack, GitHub, or anywhere that supports Markdown-style links
